## Project Overview
OceanView Reservation is a web-based reservation system developed to manage resort bookings efficiently.

## How to Run (XAMPP)
1. Copy the project folder into: `C:\xampp\htdocs\`
2. Start **Apache** (and **MySQL** if your project uses DB) from XAMPP Control Panel
3. Open your browser and visit:
   `http://localhost/oceanview-reservation/`
